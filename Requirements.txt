# ðŸš€ Setup Instructions (Java)
# -----------------------------------
1. Install Java JDK 17 or JRE 17
2. Download JavaFX SDK 17.0.14 and configure in IDE:
   - Add to `--module-path` and `--add-modules javafx.controls,javafx.fxml`
3. Install Apache Spark 3.5.4:
   - Set `SPARK_HOME` and add `$SPARK_HOME/bin` to PATH
4. (Optional) Install Python and create a virtual environment:
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
